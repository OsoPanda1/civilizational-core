## Principios de Ejecución

### No Negociables

1. **Seguridad antes que velocidad**
2. **Ética antes que crecimiento**
3. **Comunidad antes que inversores**
4. **Transparencia siempre**

### Métricas de Salud

- NPS > 40
- Churn < 5% mensual
- Margen bruto > 60%
- Zero incidentes críticos de seguridad

### Triggers de Pausa

Si cualquier métrica crítica se degrada severamente:

1. Pausar nuevos features
2. Enfocar en estabilización
3. Comunicar transparentemente
4. Reanudar cuando esté sano
